# Villa Caterina — App di gestione

App per la gestione degli appartamenti di Villa Caterina.

## Setup GitHub Pages

1. Vai su **Settings → Pages**
2. In "Source" seleziona **GitHub Actions**
3. Fai un push qualsiasi → l'app viene deployata automaticamente

L'app sarà disponibile su: `https://TUO-USERNAME.github.io/villacaterina/`
